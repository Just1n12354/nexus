# JustBetter · App

Status: konzept  ·  Version: 0.001  ·  Letztes Update: 2026-06-16

## Zweck
Persönlicher KI-Coach für Körper, Geist und Leben: ein System, das Gesundheit, Ernährung, Lernen, Gewohnheiten und persönliche Entwicklung in einer App verbindet. Kein reines Fitness-/Lern-Tool, sondern ein „digitales zweites Gehirn", das über Jahre begleitet, Fortschritte misst und individuelle Empfehlungen gibt.

## Stack / Technik (V1-Rahmen, geplant)
- Frontend: Web-App, responsive/smartphone-optimiert. Backend: Python FastAPI. DB: SQLite → später PostgreSQL.
- KI: OpenAI, Claude, Ollama lokal über Luna/Hermes.

## Stand / Funktionen
- **Reines Konzept** — im Repo nur die strukturierte Konzeptbeschreibung, kein Code, kein MVP-Schnitt, keine priorisierte Roadmap.
- Geplante Module (Denkrahmen, nicht Umsetzungszusage): Profil/Onboarding, Ernährungserfassung (Foto/Text/Barcode), Abnehm-/Fitness-Coach, Schwächen-Training (Gedächtnis/Konzentration/Englisch), KI-Tagebuch, Gewohnheiten/Streaks, Wissensspeicher, Lebensanalyse, AI Vision, Zukunftsmodul „Digital Twin".

## Betrieb
- Im PORTS.md-Schema als Konzept-Prototyp auf Port 8050 (Mac mini, stdlib http.server, via JustLauncher) vorgesehen — LaunchAgent noch anzulegen.
